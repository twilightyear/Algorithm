# [LeetCode] 9. Palindrome Number


---


### 📌 문제 링크
[LeetCode - Palindrome Number](https://leetcode.com/problems/palindrome-number/description/)


---

### ⚠️ 제약조건
- -2^31 <= x <= 2^31 - 1


---


### 🛠️ 풀이 접근 및 핵심 로직
숫자로 계산하지 않고 단순하게 문자열로 전환하여 뒤집어서 서로 비교하면 단순하지만 빠른 동작이 가능하게 구현 할 수 있다.

---


### 📊 복잡도 분석
**시간복잡도**

str(n)을 구하는데 자리수만큼 소요될것이다. O(n)
string[::-1] 을 사용하여 뒤집는데 시간이 O(n)으로 소요될 것이다.
종합적으로 최종 시간복잡도는 O(n)으로 분석해볼 수 있다.

**공간복잡도**

str(n)을 두번 연산하지 않기 위하여 string이라는 문자열에 저장했다. 그럼으로 자리수만큼 공간을 차지하게 될 것이다. O(n)
string[::-1]의 뒤집는 연산은 뒤집어진 배열을 반환한다. 즉 동일한 크기의 배열이 반환된다. O(n)
종합적으로 최종 공간복잡도는 O(n)으로 분석해볼 수 있다.

### 📝 회고 / 추후 개선점
본 문제 풀이에서는 단순 문자열의 기능을 활용하여 쉽게 풀이해보았다. 하지만 문자열이나 배열 사용을 제한한다면 숫자 그 자체만 활용하여 풀어야 할 것이다. 추후에 숫자 계산을 사용하여 도전해보겠다.
